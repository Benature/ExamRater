# Exam Rater 阅卷助手

- `src/分班小助手.py`: 根据 `班别` 区分，重新整理文件夹。
- `src/分卷小助手.py`: 
  - 根据题号区分，重新整理文件夹。并生成压缩包。
  - 在 `src/output` 文件夹下生成了相应的 `.csv` 文件。
- `html/阅卷小助手.html`: 
  - 打开后根据网页提示信息操作，导入相应 `.csv` 文件，即可开始阅卷。
